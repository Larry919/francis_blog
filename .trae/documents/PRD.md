## 1. 产品概述
Francis 的个人学术博客网站，以优雅美观的方式展示学术论文解读和个人技术分享。网站核心特色是采用 HTML5 卡片形式呈现论文解读内容，提供沉浸式阅读体验。
- 主要用途：学术成果展示、论文深度解读、个人技术品牌建设
- 目标用户：学术研究者、技术爱好者、同行学者

## 2. 核心功能

### 2.1 功能模块
1. **首页**：英雄区域、导航栏、最新论文卡片、博客文章列表、个人简介
2. **论文解读页**：HTML5 交互式论文卡片、精确定位的图表/表格、论文核心要点解读
3. **博客页**：技术文章列表、分类筛选、文章详情
4. **关于页**：个人介绍、研究方向、联系方式

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 英雄区域 | 优雅的欢迎横幅，包含个人头像、简介、快速导航 |
| 首页 | 论文精选区 | 以 HTML5 卡片形式展示最新论文解读，支持卡片悬停动画 |
| 首页 | 博客列表区 | 展示最新博客文章，带分类标签和日期 |
| 论文解读页 | 论文卡片 | 交互式 HTML5 卡片，包含论文元信息、核心贡献、精确定位的图表裁剪、关键方法解读 |
| 论文解读页 | 目录导航 | 侧边栏导航，支持快速跳转到各个解读章节 |
| 博客页 | 文章列表 | 卡片式布局，支持按标签/分类筛选 |
| 关于页 | 个人介绍 | 研究方向、发表论文、联系方式等信息 |

## 3. 核心流程
用户访问网站后，可以：
1. 浏览首页的精选论文和最新文章
2. 点击论文卡片进入详细的 HTML5 论文解读页面
3. 在论文解读页通过交互式卡片查看图表、表格和核心观点
4. 浏览博客分类，阅读技术文章
5. 通过关于页了解作者信息

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深靛蓝色系（#1e293b 深蓝灰背景 + #6366f1 靛蓝主色 + #8b5cf6 紫色点缀）
- **辅助色**：暖金色（#f59e0b）作为强调色
- **按钮风格**：圆润微立体，带微妙悬停效果和发光阴影
- **字体**：标题使用 Playfair Display（优雅衬线），正文使用 Inter（现代无衬线）
- **布局风格**：学术优雅风，卡片式布局，大量留白，精致阴影，精致边框
- **图标风格**：Lucide 线性图标，简洁精致

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | 英雄区域 | 渐变背景、微妙纹理、大尺寸优雅标题、淡入动画、头像圆形带边框 |
| 首页 | 论文卡片 | 悬浮阴影、卡片悬停上浮效果、论文缩略图、标签胶囊、精致边框 |
| 论文页 | 论文解读卡片 | HTML5 精确定位图表裁剪、分节展示、侧边锚点导航、阅读进度条 |
| 博客页 | 文章卡片 | 响应式网格、标签筛选、悬停缩放、发布日期和阅读时间 |
| 全局 | 导航栏 | 毛玻璃效果、平滑滚动、暗色/亮色模式切换 |

### 4.3 响应式设计
- 桌面优先设计
- 平板：2 列布局，调整间距
- 移动端：单列布局，汉堡菜单，优化触控体验
- 论文卡片在移动端保持良好可读性

### 4.4 动画与交互
- 页面加载时元素交错淡入（staggered reveal）
- 卡片悬停时上浮并加深阴影
- 平滑滚动导航
- 图表查看器的平滑过渡效果
- 阅读进度条随滚动更新
